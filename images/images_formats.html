Image Formats

Definitions

*PNG: PNG stands for Portable Network Graphics. This type of picture can use ver 16 million colors, which is the biggest difference between this type of image and others. This is the best format to use while programing a website because it is supported by all websites.

*JPEG: JPEG stands for Joing Photographic Experts Group. It is mainly used by digital cameras and can also supports around 16 million colors. They are best used for photographers because they are not supported by all web browsers and they use lossless compression which is the imperfect recreation of images.

*GIF: GIF stands for Graphics Interchange Format. This is a series of pictures which are put together to create the same effect as a short video that is played on repeat. They are best used for animating pictures and ideas.
